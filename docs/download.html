Vigil

Locks your Mac when you leave. Alarms if it’s moved.

A tiny two-in-one security app for Apple-silicon Macs — proximity lock over Bluetooth, plus a motion tripwire on the built-in accelerometer.

Download Vigil 1.0 View on GitHub

macOS 12+ · Apple silicon · free & open source (MIT). Motion alarm needs an M1 Pro or later.

One window. Hover any control for a plain-English explanation.

Two guards, one app
Proximity lock

Walk away, it locks

Watches your paired phone’s Bluetooth signal and locks the screen the moment you’re out of range — re-arms when you’re back. It only ever locks, never unlocks.

Motion alarm

Move it, it screams

Arms the built-in accelerometer as a tripwire. Nudge the Mac past your threshold and a siren sounds until you unlock — with a silent test mode and a safety cap.

Install — no Terminal needed
Download & unzip the file above (or grab it from the release page).
Double-click Install Vigil.command — first time, right-click › Open. It sets everything up in about a minute.
Double-click Vigil.app — pick your phone, click Arm, done.
Honest fine print
  • It never unlocks. macOS blocks apps from dismissing the login screen, so there’s no auto-unlock — use Touch ID or Apple Watch to get back in.
  • The motion alarm asks for your password or Touch ID once — click Enable (the accelerometer is root-only on macOS) — and needs an M1 Pro/Max/Ultra or newer. The proximity lock works on any Apple-silicon Mac with no password.
  • It leans on some private macOS internals. Works today; a macOS update could break parts — but it fails visibly, never silently. The README documents every fragile spot.
